# Rathan Keshav Byree — Personal Portfolio

A premium, modern, fully responsive personal portfolio website built with **HTML5**, **CSS3**, and **Vanilla JavaScript** only.

## Project Structure

```
portfolio/
├── index.html
├── css/
│   ├── main.css          # Variables, reset, base styles, components
│   ├── layout.css        # Responsive grid layouts, media queries
│   └── animations.css    # Keyframes, reveal animations, hover effects
├── js/
│   └── theme-switcher.js # Loader, navbar, observers, form, counters
├── images/
│   └── profile.jpg       # Add your profile photo here
└── README.md
```

## Setup

1. **Add your profile photo** — place a photo named `profile.jpg` inside the `images/` folder.
2. **Open in browser** — open `index.html` directly in any modern browser. No build step required.
3. **Deploy** — upload the entire `portfolio/` folder to any static host (GitHub Pages, Netlify, Vercel, etc.).

## Features

- Full-screen animated loader with glowing red bullet that shoots across the screen
- Fixed glassmorphism navbar with smooth scroll and active link highlighting
- Mobile hamburger menu with animated bars
- Hero section with CSS Grid two-column layout
- Glowing animated name with red accent highlight
- Glassmorphism code editor card with syntax highlighting
- Floating animated technology badges
- Social media icon buttons with red glow hover effects
- About section with animated stat counters (Intersection Observer)
- Skills section with animated progress bars
- Contact form with live validation
- Fully responsive — mobile-first design
- Accessible — semantic HTML5, ARIA labels, keyboard navigation, focus-visible
- Reduced motion support via `prefers-reduced-motion`
